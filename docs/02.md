# Первый сервер на Express.js 🚀

## Создание простого сервера ✨

Теперь создадим твой первый сервер! Это будет простое приложение, которое отвечает на запросы.

### Шаг 1: Создание главного файла 📄

Создай файл `app.js` в корне твоей папки:

```javascript
// Подключаем библиотеку Express
const express = require('express');

// Создаём приложение Express
const app = express();

// Говорим Express, что будем получать данные в формате JSON
app.use(express.json());

// Создаём первый маршрут (как страницу в интернете)
app.get('/', (req, res) => {
    res.json({ message: 'Привет! Сервер работает!' });
});

// Запускаем сервер на порту 4000
app.listen(4000, () => {
    console.log('Сервер запущен и работает на порту 4000');
});
```

### Шаг 2: Запуск сервера ▶️

В терминале выполни команду:
```bash
node app.js
```

В ответ ты должен увидеть:
```bash
Сервер запущен и работает на порту 4000
```

**Поздравляю! Твой первый сервер заработал! 🎉**

### Шаг 3: Тестирование сервера 🧪

1. Открой браузер
2. Перейди по адресу: `http://localhost:4000`
3. Ты увидишь: `{"message": "Привет! Сервер работает!"}`

## Маршруты 🛣️

### Что такое маршруты?

Маршрут — это как адрес в интернете. Когда ты заходишь на сайт, ты переходишь по определённому адресу (URL).

### Типы запросов

- **GET** — получить данные (например, открыть страницу) 📥
- **POST** — отправить данные (например, отправить форму) 📤
- **PUT** — обновить данные ♻️
- **DELETE** — удалить данные 🗑️

### Добавим ещё маршруты ➕

Добавь в `app.js` перед `app.listen()`:

```javascript
// маршрут для получения информации о сервере
app.get('/info', (req, res) => {
    res.json({
        name: 'Мой Express.js сервер',
        version: '0.0.1',
        status: 'работает'
    });
});

// маршрут для приветствия пользователя
app.get('/hello/:name', (req, res) => {
    const name = req.params.name;
    res.json({ message: `Привет, ${name}!` });
});

// маршрут для получения текущего времени
app.get('/time', (req, res) => {
    const now = new Date();
    res.json({
        currentTime: now.toISOString(),
        date: now.toLocaleDateString('ru-RU'),
        time: now.toLocaleTimeString('ru-RU')
    });
});
```

### Тестирование новых маршрутов:

1. `http://localhost:4000/info`
2. `http://localhost:4000/hello/Анна`
3. `http://localhost:4000/time`
